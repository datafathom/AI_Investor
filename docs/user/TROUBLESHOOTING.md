# Troubleshooting Guide

Common issues and solutions for AI Investor users.

## Login Issues

### Can't Log In

**Symptoms**: Login fails with error message

**Solutions**:
1. Verify email and password are correct
2. Check for typos (case-sensitive)
3. Click "Forgot Password" to reset
4. Clear browser cache and cookies
5. Try a different browser
6. Check if account is locked (too many failed attempts)

### Two-Factor Authentication Not Working

**Symptoms**: 2FA code not accepted

**Solutions**:
1. Ensure device time is synchronized
2. Check code hasn't expired (30-second window)
3. Generate new code from authenticator app
4. Use backup codes if available
5. Contact support to reset 2FA

## Trading Issues

### Orders Not Executing

**Symptoms**: Order placed but not executing

**Possible Causes & Solutions**:

1. **Market Closed**
   - Check market hours (9:30 AM - 4:00 PM ET)
   - Use extended hours if available

2. **Limit Price Not Reached**
   - Limit orders only execute at your price or better
   - Adjust limit price or switch to market order

3. **Insufficient Funds**
   - Check available balance
   - Verify order size
   - Wait for pending orders to settle

4. **Symbol Not Tradeable**
   - Some symbols may be restricted
   - Check symbol status

### Order Cancellation Failed

**Solutions**:
1. Order may already be executing
2. Refresh order status
3. Contact support if order stuck

## Portfolio Issues

### Portfolio Value Not Updating

**Symptoms**: Portfolio shows old values

**Solutions**:
1. Refresh the page
2. Check internet connection
3. Wait a few minutes (updates every 1-5 minutes)
4. Clear browser cache
5. Log out and log back in

### Missing Positions

**Solutions**:
1. Check order history to verify execution
2. Filter portfolio view (may be hidden)
3. Check different portfolio views
4. Contact support if position missing

### Incorrect Calculations

**Solutions**:
1. Verify all positions are included
2. Check for pending orders affecting balance
3. Review transaction history
4. Report discrepancy to support

## Performance Issues

### App Running Slowly

**Symptoms**: Pages load slowly, laggy interface

**Solutions**:
1. **Check Internet Connection**
   - Test connection speed
   - Try wired connection instead of WiFi

2. **Browser Issues**
   - Clear browser cache
   - Disable browser extensions
   - Update browser to latest version
   - Try different browser

3. **Device Performance**
   - Close other applications
   - Restart device
   - Check available memory

4. **App-Specific**
   - Reduce number of open widgets
   - Clear app data (mobile)
   - Reinstall app (mobile)

### Charts Not Loading

**Solutions**:
1. Check JavaScript is enabled
2. Disable ad blockers
3. Check browser console for errors
4. Try different browser
5. Clear browser cache

## Notification Issues

### Not Receiving Notifications

**Symptoms**: Expected notifications not arriving

**Solutions**:
1. **Check Notification Settings**
   - Go to Settings > Notifications
   - Verify notifications enabled
   - Check notification types enabled

2. **Email Notifications**
   - Check spam/junk folder
   - Verify email address is correct
   - Check email provider filters

3. **Push Notifications**
   - Check device notification settings
   - Verify app has notification permissions
   - Restart app

4. **SMS Notifications**
   - Verify phone number is correct
   - Check carrier message blocking
   - Ensure SMS plan is active

### Too Many Notifications

**Solutions**:
1. Go to Settings > Notifications
2. Adjust notification frequency
3. Disable unnecessary notification types
4. Set quiet hours
