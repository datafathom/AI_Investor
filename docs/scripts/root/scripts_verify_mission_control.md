# Script: verify_mission_control.py

## Overview
`verify_mission_control.py` is a visual integration test for the high-level application dashboards (Trader, Architect, Account).

## Core Functionality
- **Dashboard Snapshotting**: Navigates to each major "Mission Control" page and takes high-resolution screenshots to verify visual alignment, theme consistency, and the presence of critical status widgets.

## Status
**Essential (Visual Verification)**: confirms that the primary user interfaces for the major roles are correctly styled and interactive.
