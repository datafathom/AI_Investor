**Purpose**: Multi-pane command center for developers and analysts.

The Terminal Workspace is a flexible window manager that allows you to snap various widgets, charts, and terminal instances into a single view.

## Features
- **Draggable Panes**: Rearrange your workspace in real-time.
- **Widget Library**: Add specialized tools like price tickers, agent logs, or portfolio balances.
- **Direct Shell Access**: Secure, agent-monitored shell access to the underlying OS.
- **Layout Presets**: Save and load custom arrangements for different tasks.

## Getting Started
Click the '+' icon in any empty pane to add a widget. Use the snap handles at the corners to resize panes.
