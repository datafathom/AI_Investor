**Purpose**: The real-time heartbeat of the Sovereign OS.

Mission Control is the primary dashboard for monitoring the health, streams, and latencies of the entire agentic system. It provides a high-fidelity visualization of data flow between departments.

## Key Functions
- **Stream Monitoring**: View real-time Kafka topics and message throughput.
- **System Health**: Monitor CPU/Memory usage of various agents and services.
- **Latency Tracking**: Visualize end-to-end response times for critical workflows.
- **Emergency Controls**: Access to system-wide kill switches and bypasses.

## How to use
Hover over any department node to see its current status and active agents. Click on a stream to inspect the last 10 messages.
