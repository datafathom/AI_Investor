# CLI Command: deal

## Description
Private Market Deal Allocation & Waitlist Priority.

## Main Help Output
```text

Command: deal
Description: Private Market Deal Allocation & Waitlist Priority.

Available subcommands:
  allocate             Run the AUM-weighted allocation algorithm for oversubscribed deals.
  waitlist             Show the current priority waitlist for scarce deals.


```

## Subcommands

### Subcommand: `deal allocate`
Run the AUM-weighted allocation algorithm for oversubscribed deals.

#### Help Output
```text

Command: deal allocate
Description: Run the AUM-weighted allocation algorithm for oversubscribed deals.

Arguments:
  id                   Deal identifier (Required)
  capacity             Total deal capacity in dollars (Required)


```

### Subcommand: `deal waitlist`
Show the current priority waitlist for scarce deals.

#### Help Output
```text

Command: deal waitlist
Description: Show the current priority waitlist for scarce deals.


```
