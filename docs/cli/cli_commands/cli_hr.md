# CLI Command: hr

## Description
Family Office HR, Compensation & Staff Performance.

## Main Help Output
```text

Command: hr
Description: Family Office HR, Compensation & Staff Performance.

Available subcommands:
  calc-bonus           Calculate projected EOY performance bonus.
  check-comp           Show market rate benchmarking for a FO role.


```
