# CLI Command: mobility

## Description
Global Citizenship & Mobility Management.

## Main Help Output
```text

Command: mobility
Description: Global Citizenship & Mobility Management.

Available subcommands:
  check-visa           Get details on a specific Golden Visa program.
  renunciation-risk    Score the risk of renouncing citizenship.


```

## Subcommands

### Subcommand: `mobility check-visa`
Get details on a specific Golden Visa program.

#### Help Output
```text

Command: mobility check-visa
Description: Get details on a specific Golden Visa program.

Arguments:
  country              Country name (e.g. Portugal) (Required)


```

### Subcommand: `mobility renunciation-risk`
Score the risk of renouncing citizenship.

#### Help Output
```text

Command: mobility renunciation-risk
Description: Score the risk of renouncing citizenship.

Arguments:
  has-pass             True if user has replacement passport (Required)
  stability            Home country stability (0-1) (Required)


```
