# CLI Command: tests

## Description
Unified test runner for the project.

## Main Help Output
```text

Command: tests
Description: Unified test runner for the project.

Available subcommands:
  agents               Run agent-specific tests.
  all                  Run the complete test suite.
  api                  Run API endpoint tests.
  billing              Run billing service tests.
  e2e                  Run end-to-end tests.
  env                  Validate environment variables.
  frontend             Run frontend tests (vitest).
  integration          Run integration tests.
  load                 Run load/stress tests.
  schemas              Run schema validation tests.
  services             Run service layer tests.
  smoke                Run smoke tests.
  system               Run system-level tests.
  unit                 Run all unit tests.
  web                  Run web automation tests.


```
