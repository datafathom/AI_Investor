# CLI Command: vc

## Description
Venture Capital (VC) Deal Flow & Power Law Simulation.

## Main Help Output
```text

Command: vc
Description: Venture Capital (VC) Deal Flow & Power Law Simulation.

Available subcommands:
  list-deals           List active accredited investor VC deals.
  simulate             Run a VC power law simulation for a portfolio.


```

## Subcommands

### Subcommand: `vc list-deals`
List active accredited investor VC deals.

#### Help Output
```text

Command: vc list-deals
Description: List active accredited investor VC deals.

Flags:
  --min-ticket         Minimum ticket size in k (default 50) (Default: 50)


```

### Subcommand: `vc simulate`
Run a VC power law simulation for a portfolio.

#### Help Output
```text

Command: vc simulate
Description: Run a VC power law simulation for a portfolio.

Arguments:
  count                Number of investments to simulate (Optional, Default: 20)


```
