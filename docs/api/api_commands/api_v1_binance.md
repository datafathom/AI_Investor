# API Service: api_v1_binance

This document contains all endpoints registered under `api_v1_binance`.

## GET /api/v1/binance/binance/depth/<symbol>

**Summary**: Get order book depth.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
