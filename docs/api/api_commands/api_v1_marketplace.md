# API Service: api_v1_marketplace

This document contains all endpoints registered under `api_v1_marketplace`.

## POST /api/v1/marketplace/extension/<extension_id>/install

**Summary**: Install extension.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body

{
  "200": {
