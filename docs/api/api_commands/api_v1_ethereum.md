# API Service: api_v1_ethereum

This document contains all endpoints registered under `api_v1_ethereum`.

## GET /api/v1/ethereum/balance/<address>

**Summary**: Get ETH balance for an address.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
