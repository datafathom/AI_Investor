# API Service: api_v1_risk

This document contains all endpoints registered under `api_v1_risk`.

## GET /api/v1/risk/metrics/<portfolio_id>

**Summary**: Get comprehensive risk metrics for portfolio.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
