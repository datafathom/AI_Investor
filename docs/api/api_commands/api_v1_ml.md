# API Service: api_v1_ml

This document contains all endpoints registered under `api_v1_ml`.

## GET /api/v1/ml/deployment/<model_id>/performance

**Summary**: Get model performance metrics.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```
