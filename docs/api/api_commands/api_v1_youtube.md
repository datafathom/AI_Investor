# API Service: api_v1_youtube

This document contains all endpoints registered under `api_v1_youtube`.

## GET /api/v1/youtube/analyze/<video_id>

**Summary**: Analyze video transcript.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
