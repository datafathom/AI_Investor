# API Service: api_v1_alert

This document contains all endpoints registered under `api_v1_alert`.

## POST /api/v1/alert/<alert_id>/cancel

**Summary**: Cancel an alert.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
