# API Service: api_v1_web3

This document contains all endpoints registered under `api_v1_web3`.

## GET /api/v1/web3/balance/<address>/<chain>

**Summary**: Get balance for a specific wallet on a chain.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
