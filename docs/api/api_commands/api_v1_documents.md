# API Service: api_v1_documents

This document contains all endpoints registered under `api_v1_documents`.

## GET /api/v1/documents

**Summary**: List user's documents with pagination.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body

```json
{
  "200": {
    "description": "OK"
  }
}
```

| Code | Description |
| --- | --- |
| 400 | Bad Request |
