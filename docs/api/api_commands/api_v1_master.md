# API Service: api_v1_master

This document contains all endpoints registered under `api_v1_master`.

## POST /api/v1/master/bolt-proxy

**Summary**: Facilitate high-speed graph queries without exposing Neo4j credentials to the browser.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
