# API Service: api_v1_optimization

This document contains all endpoints registered under `api_v1_optimization`.

## POST /api/v1/optimization/optimize/<portfolio_id>

**Summary**: Optimize portfolio for given objective.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
