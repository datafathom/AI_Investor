# API Service: api_v1_banking

This document contains all endpoints registered under `api_v1_banking`.

## GET /api/v1/banking/accounts

**Summary**: Fetches all linked bank accounts and their balances.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
