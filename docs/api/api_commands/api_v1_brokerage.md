# API Service: api_v1_brokerage

This document contains all endpoints registered under `api_v1_brokerage`.

**Summary**: Updates/Validates brokerage API credentials.

- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body

```json
{

  }
}
```
