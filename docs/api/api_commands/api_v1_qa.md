# API Service: api_v1_qa

This document contains all endpoints registered under `api_v1_qa`.

## POST /api/v1/qa/question/<question_id>/best-answer

**Summary**: Mark best answer for question.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
