# API Service: api_v1_social

This document contains all endpoints registered under `api_v1_social`.

## GET /api/v1/social/reddit/analyze/<ticker>

**Summary**: Get sentiment analysis for a ticker.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
