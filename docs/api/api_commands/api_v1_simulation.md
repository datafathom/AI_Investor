# API Service: api_v1_simulation

This document contains all endpoints registered under `api_v1_simulation`.

## POST /api/v1/simulation/order/execute

**Summary**: Execute paper trading order.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |

---
