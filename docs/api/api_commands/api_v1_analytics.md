# API Service: api_v1_analytics

This document contains all endpoints registered under `api_v1_analytics`.

## GET /api/v1/analytics/attribution/<portfolio_id>

**Summary**: Get performance attribution for a portfolio.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
