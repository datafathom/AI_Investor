# API Service: api_v1_discord

This document contains all endpoints registered under `api_v1_discord`.

## GET /api/v1/discord/hype/<ticker>

**Summary**: Get hype score for a ticker.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
