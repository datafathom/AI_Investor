# API Service: api_v1_payment_transfer

This document contains all endpoints registered under `api_v1_payment_transfer`.

## GET /api/v1/payment_transfer/linked-vendors

**Summary**: Returns a list of financial vendors linked to the user's account.

### Details
- **Authentication**: None
- **Rate Limiting**: 100/min
- **Caching**: None

### Response Body
```json
{
  "200": {
    "description": "OK"
  }
}
```

### Error Codes
| Code | Description |
| --- | --- |
| 400 | Bad Request |
| 401 | Unauthorized |
| 500 | Server Error |
